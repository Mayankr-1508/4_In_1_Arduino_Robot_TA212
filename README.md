# 4-in-1 Arduino Robot — Smart Mobile Waste Management System

An autonomous, cost-optimized waste-collection robot designed to maintain cleanliness in indoor environments such as offices and corridors. The system integrates sensor-based navigation with robust mechanical design to automate waste collection and provide a touch-free disposal experience.

---

## Key Features

- **Autonomous Navigation** — IR sensor-based path following for efficient movement through indoor spaces
- **Smart Disposal** — UV sensor-based smart dustbin enabling hygienic, touch-free waste disposal
- **Integrated Cleaning** — Onboard vacuum mechanism for active waste removal
- **Cost-Efficient Engineering** — Optimized material selection using a mix of recycled and lab-acquired components, bringing total project cost to ₹9,071

---

## Technical Specifications

### Mechanical Assembly
- 15-component assembly consisting of GI sheet metal, 3D-printed PLA, and lathe-machined mild steel
- Integrated using custom aluminum angles and riveting

### Electronics & Control
- Developed using Arduino programming with custom sensor integration

### Manufacturing
- Precision parts manufactured through a combination of 3D printing and mechanical machining
